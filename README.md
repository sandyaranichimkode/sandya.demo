
## 📌 Project Overview

This project is a collection of interactive Power BI dashboards developed by analyzing multiple real-world datasets. The main objective is to extract meaningful insights, identify patterns, and support data-driven decision-making across different domains including healthcare, cybersecurity, e-commerce, and digital media.

The project demonstrates strong skills in data analysis, data visualization, and business intelligence using Power BI.

---

## 📁 Included Dashboards

### 1. 💊 AI-Based Drug Analysis Dashboard

This dashboard focuses on analyzing pharmaceutical data such as drug effectiveness, side effects, and patient responses.

**Key Insights:**

* Identification of high-risk drugs based on side effects
* Comparison of drug effectiveness across categories
* Trends in drug usage and patient outcomes

---

### 2. 🌐 Web Traffic & Cyber Attack Monitoring Dashboard

This dashboard analyzes network traffic and detects suspicious activities or cyber threats.

**Key Insights:**

* Detection of abnormal traffic spikes
* Identification of high-risk IP addresses
* Analysis of attack patterns (DDoS, brute-force, etc.)

---

### 3. 💻 Laptop Market Insights Dashboard

This dashboard explores laptop market trends, pricing strategies, and product performance.

**Key Insights:**

* Top-performing brands and configurations
* Price vs performance analysis
* Identification of budget-friendly and premium segments

---

### 4. 🎥 YouTube Channel Performance Dashboard

This dashboard tracks content performance and audience engagement.

**Key Insights:**

* Subscriber growth trends
* Top-performing videos based on engagement
* Impact of content type and upload frequency

---

## 🛠️ Tools & Technologies Used
